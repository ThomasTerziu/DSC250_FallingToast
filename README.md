# DSC250_FallingToast
Final Project based on the 1996 IG Nobel Prize in Physics by Robert Matthews.

Orginal Paper: https://www.researchgate.net/publication/230693562_Tumbling_toast_Murphy's_Law_and_the_fundamental_constants

Much of the concept for this was insprired by the work of Sanjoy Mahajan who revisited this "problem" in August 2022.

Mahajan's Paper: https://dspace.mit.edu/bitstream/handle/1721.1/146968/5.0121905.pdf?sequence=1

Notebook contains a simulation of buttered toast falling to the ground as apart of a final project for a course based in programming in python with minimial library imports besides math implementations. I encourage the viewer to look at the comments in the code prior to running the program.

All code contained in this notebook are my own and no generative AI was used for any of the logic. For users looking to borrow this code I ask that you credit my implementation of this project.
